Car Rental Management System

📌 Project Overview

The Car Rental Management System is a web-based application developed using HTML, CSS, and JavaScript. It allows users to browse available cars, search and filter cars by category, and book a car through a simple booking form.

✨ Features

- View available cars for rent
- Search cars by name
- Filter cars by type (SUV, Sedan, Hatchback)
- Display car details and images
- Book a car using a booking form
- Responsive and user-friendly interface

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

📂 Project Structure

car-rental-management-system/

├── index.html

├── style.css

├── script.js

└── README.md

🚀 How to Run

1. Download or clone the repository.
2. Open "index.html" in any web browser.
3. Explore available cars and test the booking functionality.
